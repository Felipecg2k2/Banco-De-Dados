# Tabelas, Registros e Campos

## O que são Tabelas

Uma **tabela** é a estrutura principal de um banco de dados.

Ela é utilizada para organizar os dados em formato de **linhas e colunas**, semelhante a uma planilha.

Cada tabela representa um tipo de informação.

Exemplos:
- tabela de usuários
- tabela de produtos
- tabela de alunos

---

## O que são Campos

Os **campos** (também chamados de colunas) representam as características dos dados armazenados na tabela.

Cada campo define um tipo de informação.

Exemplo de campos em uma tabela de usuários:
- nome
- idade
- email

---

## O que são Registros

Os **registros** (também chamados de linhas) são os dados armazenados na tabela.

Cada registro representa um item completo dentro da tabela.

Exemplo de registros:

| nome   | idade | email              |
|--------|------|--------------------|
| Felipe | 25   | felipe@email.com   |
| Ana    | 22   | ana@email.com      |

Cada linha da tabela é um registro.

---

## Relação entre Tabela, Campo e Registro

- A **tabela** organiza os dados
- Os **campos** definem os tipos de informação
- Os **registros** armazenam os dados

Ou seja:

**Tabela → Campos (colunas) + Registros (linhas)**

---

## Importância no Banco de Dados

Esses três conceitos são fundamentais porque:

- Todo banco de dados é baseado em tabelas
- Os dados são sempre armazenados em registros
- Os campos garantem organização e padronização

Sem essa estrutura, não seria possível organizar grandes quantidades de dados de forma eficiente.
