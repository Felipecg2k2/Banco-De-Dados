# Dados, Informação e Conhecimento

## O que são Dados

**Dados** são valores brutos, sem contexto ou significado completo por si só.

Eles podem ser números, textos, símbolos ou qualquer tipo de registro que represente algo.

Exemplos de dados:
- 25
- "Felipe"
- 9.5
- "Aprovado"
- 2026

Sozinhos, esses dados não explicam muita coisa.

---

## O que é Informação

**Informação** é o resultado da organização e interpretação dos dados, dando sentido a eles.

Quando os dados são contextualizados, eles passam a ter significado.

Exemplos:
- "Felipe tem 25 anos"
- "O aluno tirou nota 9.5"
- "O produto custa R$ 2026"

Agora já conseguimos entender o que está sendo representado.

---

## O que é Conhecimento

**Conhecimento** é a interpretação da informação, permitindo tomar decisões ou tirar conclusões.

Ele envolve análise, experiência e entendimento.

Exemplos:
- "Felipe é maior de idade"
- "O aluno teve um ótimo desempenho"
- "O produto é caro"

---

## Relação entre os três

Existe uma sequência lógica:

**Dados → Informação → Conhecimento**

- Dados são coletados
- Informações são organizadas
- Conhecimento é gerado a partir da interpretação

---

## Importância no Banco de Dados

O banco de dados trabalha diretamente com **dados**, mas seu objetivo final é gerar **informação útil**, que pode ser utilizada para produzir **conhecimento**.

Por isso, entender essa diferença é essencial para:
- modelar dados corretamente
- organizar informações de forma eficiente
- construir sistemas úteis
