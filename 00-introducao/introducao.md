# 💾 Banco de Dados e SQL

Bem-vindo(a) ao meu repositório de estudos em **Banco de Dados e SQL**!  
Este repositório contém conteúdos teóricos e exercícios práticos voltados para a construção de uma base sólida em modelagem de dados e linguagem SQL, começando do nível mais introdutório.

O foco é aprender de forma organizada, gradual e bem estruturada.

---

## 🎯 Objetivo

Registrar e acompanhar minha evolução no aprendizado de **Banco de Dados**, desenvolvendo:

- Base conceitual sólida
- Entendimento claro de modelagem de dados
- Domínio dos comandos fundamentais de SQL
- Organização e boas práticas

Este repositório é dedicado exclusivamente a banco de dados, sem misturar com lógica de programação ou desenvolvimento web.

---

## 🗄️ Conteúdos Abordados

Atualmente, o repositório cobre:

- O que é banco de dados
- Para que serve
- Conceitos básicos
- Estrutura de tabelas
- Modelagem simples
- Introdução ao SQL
- Comandos fundamentais:
  - `CREATE`
  - `INSERT`
  - `SELECT`
  - `UPDATE`
  - `DELETE`
- Exercícios básicos

O conteúdo será expandido gradualmente conforme o avanço nos estudos.

---

## 📚 Estrutura

Toda a organização do repositório (pastas e evolução dos conteúdos) segue uma divisão por tópicos.

Exemplo de estrutura:
00-introducao/
